Este proyecto consiste en una aplicación frontend desarrollada con Next.js, cuyo objetivo es simular una experiencia básica de e-commerce, permitiendo a los usuarios visualizar dispositivos, añadirlos a un carrito y gestionar su selección de productos.

La aplicación ha sido estructurada siguiendo buenas prácticas de desarrollo frontend, con una arquitectura clara, componentes reutilizables y manejo de estado global mediante React Context. Además, se contempla la persistencia de datos para mantener el estado del carrito incluso después de recargar la página.
